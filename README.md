## 开发环境
Ubuntu19.10

## 开发工具
vscode + Qt5.9.8 + OpenCV4.1.2

## 安装
```
git clone git@github.com:TianSHH/picop.git
cd ./picop/
qmake picop.pro
make
./bin/picop
```
